### Modulo for signals

**[%~]** performs the modulo operation on two signals. If a float is provided as an argument, the second inlet will take floats instead of signals.

### Inlets

| Inlet | Data Type    | Use      |
|:------|:-------------|:---------|
| 1     | signal       | a        |
| 2     | float/signal | b        |

### Outlets

| Outlet | Data Type | Return    |
|:-------|:----------|:----------|
| 1      | signal    | remainder |

### Arguments

| Arg | Data Type | Use                  |
|:----|:----------|:---------------------|
| 1   | float     | value to modulo with |

### Alias 

mod~
